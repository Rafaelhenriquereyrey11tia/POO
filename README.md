Projeto de POO: Sistema de Impressão

Objetivo

Criar um sistema em Java que permita a comunicação com impressoras Elgin, simulando operações de um caixa (PDV) através de um menu interativo.


---

Materiais Necessários

Impressora

Bobina da impressora

Gaveta para teste

IntelliJ IDEA Community Edition 2024.3.5

JDK instalado

Arquivo da biblioteca da impressora (DLL)


Link do arquivo utilizado:
https://elgindevelopercommunity.github.io/group___m1.html#ga928f0795631b062f8d5c8c20b9681d8d


---

Como Baixar o Projeto

1. Acesse o repositório no GitHub


2. Clique no botão Code (botão verde)


3. Clique em Download ZIP


4. Vá até a pasta Downloads


5. Clique com o botão direito no arquivo ZIP


6. Clique em Extrair Tudo


7. Clique em Extrair e aguarde completar
---

Como Executar o Sistema

1. Abra o IntelliJ IDEA


2. Abra o projeto extraído


3. Abrir a plataforma responsável pela impressora antes de executar

Caso contrário, a impressora não funcionará

4. Alterar o local do arquivo DLL no código:

Linha onde está o caminho da DLL:

"C:\\Users\\rey_rafael\\Downloads\\Java-Aluno EM\\Java-Aluno EM\\Java-Aluno EM\\E1_Impressora01.dll"

Substituir pelo caminho correto do usuário do computador, por exemplo:

"C:\\Users\\SeuUsuario\\Downloads\\E1_Impressora01.dll"

Se o caminho não for alterado, o programa não consegue localizar o arquivo necessário para comunicação com a impressora.

5. Após configurar, clicar em Run (Executar)


6. Escolher as opções do menu e utilizar normalmente


7. Ao finalizar, encerrar pelo menu
